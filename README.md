# Ceylon Brew Café Landing Page

A modern, responsive landing page for Ceylon Brew Café, a specialty coffee shop in Maharagama.

## Features
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **Fast Loading**: Optimized images and minimal code.
- **Interactive Elements**: Mobile menu, lightbox gallery, smooth scrolling.
- **Local Vibe**: Designed to reflect the warm, Sri Lankan coffee culture.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with Flexbox/Grid, CSS Variables, and responsive media queries.
- **JavaScript**: Vanilla JS for interactivity (no heavy frameworks).
- **Assets**: High-quality images from Unsplash.

## Installation & Setup
1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. No build process required (Vanilla JS/CSS).
